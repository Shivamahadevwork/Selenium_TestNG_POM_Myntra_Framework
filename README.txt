Key Notes:

This automation framework utilizes Java as the programming language and leverages Selenium
to enable web automation for Myntra web application's features. TestNG is the chosen unit testing framework.
The framework is constructed with a combination of Page Object and Page Factory design patterns to enhance code modularity and maintainability.

To execute the framework, you will need the following software, tools, and dependencies:

*Java
*Maven
*Selenium
*TestNG

* Tests can be executed either by running the Regression.xml file or running the 
AddProdToWLTest class directly or running in cmd line (using --> MVN TEST command).

* To generate and serve the reports in browser
	* Use allure serve allure-results
	
* The screenshots of the tests are stored under Screenshots folder

* The test recording is stored under test-recordings folder (suggest using VLC Media Player)
